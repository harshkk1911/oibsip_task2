Altered Fingerprint Detection

With the help of two databases, one for modified fingerprints and one for modified original fingerprints, we hope to create a fingerprint identification system that can compare the modified fingerprints to the originals. The original fingerprint database will contain intact fingerprints, but modified fingerprints will contain fingerprints that have undergone various modifications such as blurring, distortion, or noise in the database. We extract features from input fingerprints and compare them with features extracted from two databases using advanced machine learning algorithms and deep neural networks. The scanned fingerprint will be correctly matched with the original by our algorithm, which will determine the trust score for the match.
